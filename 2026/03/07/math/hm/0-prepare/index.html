1.1.1.1.2 幂函数

$y \ : = \ : x ^ { \mu }$ ， $\mu$ 为实数，当 $x \ > \ 0$ ，$y = x ^ { \mu }$ 都有定义：

对于幂函数可以根据不同幂下相同单调性来研究最值：

  1. $\sqrt { u } \ \sqrt [ 3 ] { u }$ 可以使用 $u$ 来研究。

  2. $| u |$ 可以使用 $u ^ { 2 }$ 来研究。

  3. $\frac { 1 } { u } , u > 0$ 可以使用 $u$ 来研究，但是最值相反。

  4. $u _ { 1 } u _ { 2 } . . . u _ { n }$ 可以使用 $\textstyle \sum _ { i = 1 } ^ { n } \ln u _ { i }$ 来研究。

1.1.1.1.3 指数函数

指数函数具有如下性质：

$$
y = a ^ {x} (a > 0, a \neq 1):
$$

  1. 特殊函数值： $a ^ { 0 } = 1$ 。

  2. 定义域： $( - \infty , + \infty )$ ，值域： $( 0 , + \infty )$ 。

  3. 单调性： $a \ > \ 1$ ， $\boldsymbol { y } \ : = \ : \boldsymbol { a } ^ { x }$ 单调增，$0 < a < 1$ ， $y = a ^ { x }$ 单调减。

  4. 常用指数函数： $y = e ^ { x }$ 。

  5. 极限：$\lim_{x \to -\infty} e^x = 0$, $\lim_{x \to +\infty} e^x = +\infty$。

$y = l o g _ { a } x ( a > 0 , a \neq 1 )$ 为 $y = a ^ { x }$ 的反函数：

常用公式： $x = e ^ { \ln x }$ ， $u ^ { v } = e ^ { \ln u ^ { v } } =$ $e ^ { v \ln u } ( x > 0 , u > 0 )$ 。

对数函数具有如下性质：

  1. 特殊函数值： $\log _ { a } 1 = 0$ ， $\mathit { l o g } _ { a } a = 1 , \ln 1 = 0 , \ln e = 1 { \mathrm { ~ } }$

  2. 定义域： $( 0 , + \infty )$ ，值域： $( - \infty , + \infty )$ 。

  3. 单调性： $a > 1$ ， $y = \log _ { a } x$ 单调增， $0 < a < 1$ ， $y = \log _ { a } x$ 单调减。

  4. 常用对数函数： $y = \ln x$ ， $e = 2 . 7 1 8 2 8 . . .$。

  5. 极限：$\operatorname* { l i m } _ { x \to 0} \log _ { a } x = - \infty$ ，$\operatorname* { l i m } _ { x \to 0} \log _ { a } x = + \infty$。

1.2 极坐标系图像

1.2.1 极坐标系

1.2.1.1 极坐标定义

• 极点：平面内的一个定点 $O$ 。

• 极轴：自极点引出的射线 $O x$ 。

• 极坐标系：选定长度单位、角度单位与正方向（通常为逆时针）建立的坐标系。

• 极径：设 $M$ 为屏幕一点，极点 $O$ 与 $M$ 的距离 $| O M |$ ，记为 $\rho$

• 极角：以极轴 $O x$ 为始边，射线 $O M$ 为终边的角 $x O M$ 为 $M$ 的极角，记为 $\theta$ 。

• 极坐标：有序数对 ρ、 $\theta$ 为 $M$ 的极坐标，记为 $M ( \rho , \theta )$

1.2.1.2 极坐标系转换

设 $M$ 为平面一点，直角坐标为 $( x , y )$ ，极坐标 $( \rho , \theta )$ ，其关系是：

$$
\begin{cases}
x = \rho \cos \theta \\
y = \rho \sin \theta
\end{cases}, \quad
\begin{cases}
\rho^2 = x^2 + y^2 \\
\tan \theta = \frac{y}{x} (x \neq 0)
\end{cases}
$$

1.2.1.3 常用极坐标方程

直线的极坐标方程：

• 从极点 $O$ 发出的一条射线： $\tan \theta = k$ （由于 $k$ 不知道符号所以不能直接转换为反三角函数）。

• 过点 $( a , 0 )$ 且垂直于极轴的直线的极坐标方程： $\rho = a \sec \theta = { \frac { a } { \cos \theta } }$ 。

• 过点 $\left( a , { \frac { \pi } { 2 } } \right)$ 且平行于极轴的直线的极坐标方程： $\rho = a \csc \theta = { \frac { a } { \sin \theta } }$ 。 圆的极坐标方程：

• 圆心为极点，半径为 $r$ 的圆的极坐标方程： $\rho = r$ 。

• 圆心 $O ^ { \prime } ( r , 0 )$ ，半径为 $r$ 的圆的极坐标方程： $\rho = 2 r \cos \theta$ 。

• 圆心 $O ^ { \prime } \left( r , { \frac { \pi } { 2 } } \right)$ ，半径为 $r$ 的圆的极坐标方程： $\rho = 2 r \sin \theta$ 。

抛物线的极坐标方程：

• $y = a x ^ { 2 }$ 的极坐标方程： $\rho = { \frac { 1 } { a } } \tan \theta \sec \theta$ 。

• $y ^ { 2 } = a x$ 的极坐标方程： $\rho = a \cot \theta \csc \theta$ 。

1.2.2.1 心形线（外摆线）

心形线又称为心脏线，表示是一个圆上的固定一点在它绕着与其相切且半径相同的另外一个圆周滚动时所形成的轨迹。

1.3 参数法

如果很难使用直角坐标或极坐标来表示曲线，那么可以引入一个新的变量参数来表示，即得到参数方程：

$$
\begin{cases}
x = x(t) \\
y = y(t)
\end{cases}
$$

1.3.1 摆线（平摆线）

摆线，又称旋轮线、圆滚线，是一个圆沿一条直线滚动时，圆边界上一定点所形成的轨迹。

令圆半径为 $r$ ，摆点与圆心所成直线所转动夹角对应弧度为 $t$ ，其中 $t \in [ 0 , 2 \pi ]$ ，

$$
\text{所对应参数方程为：} \begin{cases}
x = r(t - \sin t) \\
y = r(1 - \cos t)
\end{cases}
$$

1.3.2 星形线（内摆线）

与半径为 $r$ 的定圆内切的半径为 $\frac { r } { 4 }$ 的动圆沿定圆无滑动地滚动，动圆上一点的轨迹称为星形线。

$$
\text{程：} \begin{cases}
x = r \cos^3 t \\
y = r \sin^3 t
\end{cases}
$$

由 $\cos ^ { 2 } t + \sin ^ { 2 } t = 1$ 得到直角坐标方程：

$$
x ^ {\frac {2}{3}} + y ^ {\frac {2}{3}} = r ^ {\frac {2}{3}} 。
$$

2.3.1 反正弦函数

正弦函数 $y = \sin x$ 在主值区间 $\left[ - \frac { \pi } { 2 } , \frac { \pi } { 2 } \right]$ 上的反函数就是反正弦函数，记为$y = \arcsin x$ 或 $y = \sin ^ { - 1 } x$ 。表示其区间上正弦值等于 $x$ 的一个角。

反正弦函数与正弦函数图像一样都是关于原点对称、严格单调递增、有界的奇函数。

当 $x \in [ - 1 , 1 ]$ 时， $\arcsin ( - x ) = - \arcsin x$ 。

$x \in \left[ - \frac { \pi } { 2 } , \frac { \pi } { 2 } \right]$ 时， $\arcsin ( \sin x ) = x$ 。

当 $x \in [ - 1 , 1 ]$ 时， $\sin ( \arcsin x ) = x$ 。

2.3.2 反余弦函数

余弦函数 $y = \cos x$ 在主值区间 $[ 0 , \pi ]$ 上的反函数就是反余弦函数，记为$y = \operatorname { a r c c o s } x$ 或 $y = \cos ^ { - 1 } x$ 。表示其区间上余弦值等于 $x$ 的一个角。

反余弦函数是严格单调递减、有界的非奇非偶函数，图像关于 $\left( 0 , { \frac { \pi } { 2 } } \right)$ 对称。

因为关于 $\left( 0 , { \frac { \pi } { 2 } } \right)$ 对称，若 $x _ { 1 } , x _ { 2 } \ \in \ ( - 1 , 1 )$ ， $x _ { 1 } ~ = ~ - x _ { 2 }$ ，则 arccos $x _ { 1 } +$ arccos $x _ { 2 } = \pi$ 。

当 $x \in ( - 1 , 1 )$ 时， $\operatorname { a r c c o s } ( - x ) + \operatorname { a r c c o s } x = \pi$ 。

当 $x \in [ 0 , \pi ]$ 时， $\operatorname { a r c c o s } ( \cos x ) = x$ 。

当 $x \in [ - 1 , 1 ]$ 时， $\cos ( \operatorname { a r c c o s } x ) = x$ 。

当 $x \in [ - 1 , 1 ]$ 时， $\sin ( \operatorname { a r c c o s } x ) = { \sqrt { 1 - x ^ { 2 } } }$ 。令 $u = \operatorname { a r c c o s } x \in [ 0 , \pi ]$ ，所以cos $u = x$ ，从而 $\sin ( \operatorname { a r c c o s } x ) = \sin u = { \sqrt { 1 - \cos ^ { 2 } u } } = { \sqrt { 1 - x ^ { 2 } } } \quad$ 。

同理可得 $\cos ( \arcsin x ) = { \sqrt { 1 - x ^ { 2 } } }$ 。

$\arcsin x + \operatorname { a r c c o s } x = { \frac { \pi } { 2 } }$ 。证明需要分三种情况。

2.3.3 反正切函数

正切函数 $y = \tan x$ 在主值区间 $\left[ - \frac { \pi } { 2 } , \frac { \pi } { 2 } \right]$ 上的反函数就是反正切函数，记为$y = \arctan x$ 或 $y = \tan ^ { - 1 } x$ 。表示其区间上正切值等于 $x$ 的一个角。

正切函数是关于原点对称、严格单调递增、有界的奇函数。值域为 $\left( - { \frac { \pi } { 2 } } , { \frac { \pi } { 2 } } \right)$ ，定义域为 $( - \infty , + \infty )$ 。

当 $x \in ( - \infty , + \infty )$ 时， $\arctan ( - x ) = - \arctan x$ 。

当 $x \in \left( - \frac { \pi } { 2 } , \frac { \pi } { 2 } \right)$ ， $\arctan x ) = x$。

例题：求 $\arctan {\frac { 1 } { 2 } } + \arctan { \frac { 1 } { 3 } }$ 的值。

解：由于求反正切函数不方便，所以转换为正切函数来求。

令$\arctan {\frac { 1 } { 2 } } = \alpha$ ， $\arctan {\frac { 1 } { 3 } } = \beta$ ，所以 $\tan \alpha = { \frac { 1 } { 2 } }$ ，$\tan \beta = \frac { 1 } { 3 }$ 。

即 $\alpha + \beta = { \frac { \pi } { 4 } } = \arctan { \frac { 1 } { 2 } } + \arctan { \frac { 1 } { 3 } }$ 。

2.5 对数运算法则

重点：

  1. $\log _ { a } ( M N ) = \log _ { a } M + \log _ { a } N$ （积的对数 $=$ 对数的和）。

  2. $\log _ { a } ( \frac { M } { N } ) = \log _ { a } M - \log _ { a } N$ （商的对数 $=$ 对数的差）。

  3. $\log _ { a } M ^ { n } = n \log _ { a } M$ （幂的对数 $=$ 对数的倍数）。

  4. $\log _ { a } \sqrt [ n ] { M } = \frac { 1 } { n } \log _ { a } M$ 。

所以以后多项相乘相除乘方开方都取对数进行化简。

对于分数相减的对数先通分再进行对数减法。

如下面这个题（先不要求能直接证明）：

例题：证明 $\frac { 1 } { x + 1 } < \ln ( 1 + \frac { 1 } { x } ) < \frac { 1 } { x }$ ，其中 $x > 0$ 。

证明：首先因为证明中间项无法进行直接处理，又看到是一个对数，所以进行通分： $\ln ( 1 + { \frac { 1 } { x } } ) = \ln { \frac { x + 1 } { x } } = \ln ( x + 1 ) - \ln x$ 。

又因为是证明该中间式在一个区间，所以很明显会想到拉格朗日中值定理：$f ( b ) - f ( a ) = f ^ { \prime } ( \xi ) ( b - a )$ 。

得到原式 $= f ^ { \prime } ( \xi ) = ( \ln \xi ) ^ { \prime } = \frac { 1 } { \xi }$ 1 又中值定理下 $a < \xi < b$ 且 $x > 0$ ，所以$\frac { 1 } { b } < \frac { 1 } { \xi } < \frac { 1 } { a }$ ， 得到 $0 < \frac { 1 } { x + 1 } < \frac { 1 } { \xi } < \frac { 1 } { x }$ 。

所以原式 $\frac { 1 } { x + 1 } < \ln ( 1 + \frac { 1 } { x } ) < \frac { 1 } { x }$ 成立。

2.7 因式分解公式

重点为 3、4、7 和 11 的公式。

  1. $( a + b ) ^ { 2 } = a ^ { 2 } + 2 a b + b ^ { 2 }$ 。

  2. $( a - b ) ^ { 2 } = a ^ { 2 } - 2 a b + b ^ { 2 }$ 。

  3. $( a + b ) ^ { 3 } = a ^ { 3 } + 3 a ^ { 2 } b + 3 a b ^ { 2 } + b ^ { 3 }$ 。

  4. $( a - b ) ^ { 3 } = a ^ { 3 } - 3 a ^ { 2 } b + 3 a b ^ { 2 } - b ^ { 3 }$ 。

  5. $a ^ { 2 } - b ^ { 2 } = ( a + b ) ( a - b )$ 。

  6. $a ^ { 3 } - b ^ { 3 } = ( a - b ) ( a ^ { 2 } + a b + b ^ { 2 } )$ 。

  7. $a ^ { 3 } + b ^ { 3 } = ( a + b ) ( a ^ { 2 } - a b + b ^ { 2 } )$ 。

  8. $n$ 为正整数时， $a ^ { n } - b ^ { n } = ( a - b ) ( a ^ { n - 1 } + a ^ { n - 2 } b + \cdot \cdot \cdot + a b ^ { n - 2 } + b ^ { n - 1 } )$ 。

  9. $n$ 为正偶数时， $a ^ { n } - b ^ { n } = ( a + b ) ( a ^ { n - 1 } - a ^ { n - 2 } b + \cdot \cdot \cdot + a b ^ { n - 2 } - b ^ { n - 1 } )$ 。

  10. $n$ 为正奇数时， $a ^ { n } + b ^ { n } = ( a + b ) ( a ^ { n - 1 } - a ^ { n - 2 } b + \cdot \cdot \cdot - a b ^ { n - 2 } + b ^ { n - 1 } )$ 。

11.二项式定理：$(a + b)^n = \sum_{k=0}^{n} C_n^k a^{n-k} b^k = a^n + na^{n-1}b + \frac{n(n-1)}{2!} a^{n-2} b^2 + \dots + \frac{n(n-1) \dots (n-k+1)}{k!} a^{n-k} b^k + \dots + nab^{n-1} + b^n$

对于二项式定理需要记忆，后面的幂比较简单，而前面的系数比较困难，可以使用杨辉三角形来记忆：

$$
\begin{array}{c c c c c} & & & C _ {0} ^ {0} \\ & & & C _ {1} ^ {0} & C _ {1} ^ {1} \\ & & C _ {2} ^ {0} & C _ {2} ^ {1} & C _ {2} ^ {2} \\ & C _ {3} ^ {0} & C _ {3} ^ {1} & C _ {3} ^ {2} & C _ {3} ^ {3} \\ C _ {4} ^ {0} & C _ {4} ^ {1} & C _ {4} ^ {2} & C _ {4} ^ {3} & C _ {4} ^ {4} \end{array}
$$

2.8 阶乘与双阶乘

  1. $n ! = 1 \times 2 \times 3 \times \cdots \times n$ ，其中 $0 ! = 1$ 。

  2. $(2n)!! = 2 \times 4 \times 6 \times \dots \times (2n) = 2^n \cdot n!$。

  3. $( 2 n - 1 ) ! ! = 1 \times 3 \times 5 \times \cdots \times ( 2 n - 1 )$。

以后的华里士公式（点火公式，在定积分中）会使用到，如下面的题目：

例题：计算 $\begin{array} { r } { \int _ { 0 } ^ { \frac { \pi } { 2 } } \sin ^ { 1 0 } x \mathrm { d } x } \end{array}$ 与 $\int _ { 0 } ^ { \frac { \pi } { 2 } } \cos ^ { 9 } x \mathrm { d } x$

解：原式 1 为偶数次幂，所以 $= { \frac { 9 } { 1 0 } } \cdot { \frac { 7 } { 8 } } \cdot { \frac { 5 } { 6 } } \cdot { \frac { 3 } { 4 } } \cdot { \frac { 1 } { 2 } } \cdot { \frac { \pi } { 2 } } = { \frac { \pi } { 2 } } \cdot { \frac { 9 ! ! } { 1 0 ! ! } }$

原式 2 为奇数次幂，所以 $= { \frac { 8 } { 9 } } \cdot { \frac { 6 } { 7 } } \cdot { \frac { 4 } { 5 } } \cdot { \frac { 2 } { 3 } } = { \frac { 8 ! ! } { 9 ! ! } }$

2.9 常用不等式

非常重要。

2.9.1 绝对值不等式

若 $a$ ， $b$ 为实数，则：

  1. $| a \pm b | \leqslant | a | + | b | ,$ 。

  2. 推广公式一到离散区间： $\left| a _ { 1 } \pm a _ { 2 } \pm a _ { 3 } \pm \cdot \cdot \cdot \pm a _ { n } \right| \leqslant \left| a _ { 1 } \right| + \left| a _ { 2 } \right| + \cdot \cdot \cdot + \left| a _ { n } \right|$。

  3. 推广公式一到连续区间且 $f ( x )$ 在 $[ a , b ] ( a \ < \ b )$ 上可积： $\begin{array} { r } { | \int _ { a } ^ { b } f ( x ) \mathrm { d } x | \ \leqslant } \end{array}$ $\textstyle \int _ { a } ^ { b } | f ( x ) | \mathrm { d } x$ 。因为符号不一定相同的面积代数和一定小于同为正的面积代数和。

  4. $| | a | - | b | | \leqslant | a - b |$ 。后式子为两点之差，前式子可以看作 $a , \ b$ 两点与 0 之间的距离的差，若异号则两者必然抵消一部分，若同号则就等于后式。

2.9.2 根号不等式

公式一非常重要，即算数平均值大几何平均值。

$$
a, b, c > 0:
$$

  1. ${ \sqrt { a b } } \leqslant { \frac { a + b } { 2 } } \leqslant { \sqrt { \frac { a ^ { 2 } + b ^ { 2 } } { 2 } } }$。

  2. ${ \sqrt [ 3 ] { a b c } } \leqslant { \frac { a + b + c } { 3 } } \leqslant { \frac { a ^ { 2 } + b ^ { 2 } + c ^ { 2 } } { 3 } }$。

例题：证明函数 $f ( x ) = { \frac { x } { 1 + x ^ { 2 } } }$ 在 $( - \infty , + \infty )$ 内有界。

证明：可以使用极限，若极限存在则函数有界，这里使用有界性定义与不等式来完成。

  1. 当 $x = 0$ 时， $f ( 0 ) = { \frac { 0 } { 1 } }$ ，有界。

  2. 当 $x \neq 0$ 时，原式分式上下都有 $x$ ，所以简化公式： $f ( x ) = { \frac { \frac { x } { x } } { { \frac { 1 + x ^ { 2 } } { x } } } } = { \frac { 1 } { { \frac { 1 } { x } } + x } }$。

∵ 需要证明有界性，以及根号不等式下需要参数大于 0，所以需要证明$|f(x)| = \frac{1}{\frac{1}{|x|} + |x|} \leqslant M$。

$\because \frac{a+b}{2} \geqslant \sqrt{ab}, \therefore \frac{\frac{1}{|x|} + |x|}{2} \geqslant \sqrt{\frac{1}{|x|} \cdot |x|} = 1$。

$$
\therefore | f (x) | = \frac {1}{\frac {1}{| x |} + | x |} \leqslant \frac {1}{2} 。
$$

故整个函数在 $R$ 上有界。
